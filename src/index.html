A Retrieval-Augmented Generation system for querying real vulnerability data from the National Vulnerability Database (NVD). Instead of relying on generic LLM knowledge, questions are answered by finding the 5 most semantically relevant CVEs and grounding Llama 3.1's response in actual exploit data — across Apache, OpenSSL, Linux, WordPress, and more.

An unsupervised ML pipeline that classifies network hosts by risk profile — Critical, High, Moderate, or Low — using K-Means clustering over aggregated vulnerability indicators including CVSS scores, EPSS probabilities, CISA KEV flags, and time-to-exposure. Outputs a labeled CSV and PCA scatter visualization.
